# Immaybesorry

## Getting started

```
$ git clone https://github.com/Cadiac/immaybesorry.git
$ cd immaybesorry

$ python -m venv immaybesorry
$ pip install -r requirements.txt
$ python scripts/migrate.py

$ python -m imneversorry.py
```
